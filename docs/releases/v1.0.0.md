## Technical Highlights

### Architecture

- **Backend:** NestJS (TypeScript) with modular architecture
- **Frontend:** Next.js 14 with App Router and React Server Components
- **Database:** PostgreSQL 16 with pgvector extension
- **Cache:** Redis 7 for sessions and rate limiting
- **ORM:** Prisma with type-safe database access

### Security

- **Authentication**
  - JWT-based authentication with refresh tokens
  - OAuth 2.0 with PKCE flow
  - AES-256-GCM encryption for sensitive fields
  - bcrypt password hashing (12 rounds)

- **Data Protection**
  - TLS 1.3 for data in transit
  - AES-256-GCM for data at rest
  - Row-Level Security (RLS) for multi-tenancy
  - CSRF protection
  - Input validation and sanitization

- **Compliance**
  - GDPR compliant (EU data residency)
  - Data export functionality (Subject Access Request)
  - Data deletion (Right to Erasure)
  - Audit logging (7-year retention)
  - Cookie consent management

### Performance

- **Optimizations**
  - Database query optimization with indexes
  - Connection pooling (PgBouncer)
  - Redis caching for frequent queries
  - Lazy loading and pagination
  - Image optimization and CDN

- **Monitoring**
  - Prometheus metrics
  - Sentry error tracking
  - Health check endpoints
  - Performance monitoring
  - Audit logs

### Testing

- **Comprehensive Test Coverage**
  - 211+ end-to-end tests (Playwright)
  - Unit tests with Jest
  - Integration tests
  - 98%+ code coverage
  - Automated CI/CD testing

## Known Issues

### Minor Issues

1. **Safari Browser Compatibility**
   - Issue: Semantic search may be slower on Safari
   - Workaround: Use Chrome or Firefox for best performance
   - Status: Fix planned for v1.1.0

2. **Large Import Performance**
   - Issue: Importing 1000+ contacts may take 5-10 minutes
   - Workaround: Import in batches or use background import
   - Status: Performance optimization in progress

3. **Email Sync Delay**
   - Issue: Email interactions may take up to 5 minutes to appear
   - Workaround: Manually trigger sync from integrations page
   - Status: Working as designed (rate limit considerations)

### Limitations

- **Contact Limit:** Free plan limited to 100 contacts
- **File Upload:** Maximum 10MB per file
- **API Rate Limit:** 100 requests per minute per user
- **Export Format:** Currently supports JSON and CSV (VCF coming in v1.1)

## Installation & Setup

See [Admin Guide](/docs/admin/ADMIN_GUIDE.md) for detailed installation instructions.

**Quick Start:**

```bash
# Clone repository
git clone https://github.com/dxheroes/pmcrm.git
cd pmcrm

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your configuration

# Setup database
npm run db:setup

# Start development server
npm run dev
```
